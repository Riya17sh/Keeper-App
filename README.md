Keeper App is a simple, intuitive note-taking application inspired by a notepad, designed to help users quickly capture and manage their thoughts in a structured way. 
It provides a clean interface with a fixed-format input system, ensuring consistency and ease of use. The App is built to strengthen core frontend development concepts 
such as component-based architecture, state management, event handling, and reusable UI components, while delivering a practical and user-centric application.

Features:
Add notes using a fixed format consisting of a Title and Description, making notes easy to read and organize.
A compact "Add" button allows users to instantly save notes without unnecessary steps.
Each note comes with a Delete button, enabling users to remove notes effortlessly whenever they are no longer needed.
Focused on simplicity, the app offers a clutter-free experience for quick note management.

How It Works:
1. Enter a Title and Description in the input fields.
2. Click the Add button to save the note.
3. View the note displayed instantly on the screen.
4. Click the Delete button on any note to remove it.

Use Cases
1. Daily task reminders
2. Quick idea capture
3. Study or learning notes
4. Lightweight personal note management
